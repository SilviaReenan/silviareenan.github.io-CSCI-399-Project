Results

Channel open and funded

The start of the shows the two pings failing due to the lack of connection to the internet. Then the transaction is done directly after, and is completed without failure. Proving that transactions between two nodes while not connected to the internet is possible and can be done successfully through the mesh network.

Summary

The 3 nodes were connected via a mesh network, and through this we were able to take 2 of the nodes offline, no internet connection but still connected to each other. Then establish a channel between them and fund it completely off the internet using the third node on the mesh network. This third node handles the internet and all the transaction information allowing the two nodes to speak to the internet through it and establish connection.
